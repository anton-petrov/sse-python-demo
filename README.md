### Setup
# Demo SSE streaming data in python
## Prerequisites
Python 3.9+, Poetry package manager.

1. poetry shell
2. poetry install
3. chmod +x ./start.sh

#### Demo
- Run server: `./start.sh`
- Run client in browser `http://localhost:8000`
- Run client in terminal `python app/client.py` or `poetry run app/client.py`
